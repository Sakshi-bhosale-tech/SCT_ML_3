# Cat vs Dog Image Classification using SVM

This project is part of my Machine Learning Internship at SkillCraft Technology.

## 📌 Project Overview
The goal of this project is to classify images of cats and dogs using a Support Vector Machine (SVM) algorithm.

The model is trained on image data and predicts whether a given image belongs to a cat or a dog.

---

## 🧠 Algorithm Used
- Support Vector Machine (SVM)
- Supervised Machine Learning

---

## 📊 Dataset
- Dogs vs Cats Dataset (Kaggle)
- The dataset contains labeled images of cats and dogs.
- Link :- https://www.kaggle.com/c/dogs-vs-cats/data

---

## ⚙️ Steps Performed
1. Imported required libraries  
2. Loaded image dataset from local directory  
3. Assigned labels based on filename (cat = 0, dog = 1)  
4. Converted images to grayscale  
5. Resized images to 64x64 pixels  
6. Flattened images into feature vectors  
7. Split data into training and testing sets  
8. Trained SVM model using linear kernel  
9. Predicted labels on test data  
10. Evaluated model accuracy  
11. Tested model on custom images  

---

## 📈 Results
- The model achieved good accuracy in distinguishing between cats and dogs.
- Successfully predicted labels for new unseen images.
